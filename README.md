# scenery only



## サイト概要


### サイトテーマ

風景写真を投稿したり、投稿された写真を閲覧、検索できるSNSサービス



### テーマを選んだ理由

風景写真が好きで、自分の撮った写真を共有したり、他人の撮った写真を見て実際に見に行きたいと思ったため。



### ターゲットユーザ

風景写真が好きな人



### 主な利用シーン

撮影した写真を共有したいとき。見たことない風景を検索したいとき。
